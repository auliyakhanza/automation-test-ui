**Katalon Automation Framework - Setup**


**Overview**

This project is automated tests built using Katalon Studio for Web UI Testing.


**Prerequisites**

Java JDK 8 or 11

Katalon Studio (latest stable version recommended)

Google Chrome/Microsoft Edge/Firefox

**Setup**
1. Open Katalon Studio
2. Click Open Project
3. Select project folder


**Run Test**
1. Run Test Case: Right click → Run
2. Run Test Suite: Open → Run
